# CV Builder (HTML/CSS/JS)

Professional CV builder website.

## Features
- Multi-section form (separate input fields)
- Live preview (Professional look)
- Optional profile photo (stored locally in your browser)
- Multiple download options:
  - **Download PDF** via browser print (Save as PDF)
  - **Download Word (.doc)** generated locally as HTML-based Word file
- Theme + layout options
